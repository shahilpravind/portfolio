# Datec Fiji In-House Inventory Management Android and Web App (Dec 2016 Project)
The code and files for this project cannot be released since I do not have authorization to do so as they are the private property of Datec Fiji PTE Limited. However, images of the app is provided below.


## Project Information
1. Android app developed using Java and the Android SDK.
2. Web app developed using PHP, MySQL, HTML, CSS, JavaScript, AJAX.
